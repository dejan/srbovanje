= srbovanje

Ruby gem za transliteraciju ćirilice na latinicu i tzv. ASCII latinicu


== Upotreba

  >> "Ђурђевак".recode_sr_latin
  => "Đurđevak"

  >> "Железница Србије".recode_sr_ascii_latin
  => "Zeleznica Srbije"
